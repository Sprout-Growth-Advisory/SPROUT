# Sprout Growth Systems

One-page landing page for Sprout Growth Systems.

## Files

- `index.html` - landing page content
- `styles.css` - responsive visual styling
- `script.js` - small browser behavior
- `assets/` - website images and logo assets

## Open Locally

Open `index.html` in a browser.

## Publish With GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open the repository settings.
3. Go to **Pages**.
4. Set the source to the `main` branch and the root folder.
5. Save. GitHub will provide the public site URL.
